Adding content with consistent padding can be done easily by using the Building block components.

## Basic usage

Import the component and nest the content in the CardContent component.

```js
import { Card, CardActions, Button } from "ubiq-ui"

<Card>
  <CardActions>
    <Button>Learn more</Button>
  </CardActions>
</Card>
```
