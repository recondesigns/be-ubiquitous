## Basic usage

Import the component, and place it in the UI.

```js
import { Button } from "ubiq-ui"

<Button>
  Send
</Button>
```

### Button type

Apply rounded or straight edges to the Button.

<Canvas of={ButtonStories.ButtonType} />

### Size

Optionally, the size of the button can be set using the `size` prop.

<Canvas of={ButtonStories.Size} />

### Full width

By default, the width grows with the content of the label. Pass the `isFullWidth` prop to force the component's width to 100% of the parent container.

<Canvas of={ButtonStories.FullWidth} />

### Icon

Pass an icon to the `iconLeft` prop to render a Button with a preceding icon.

<Canvas of={ButtonStories.LeftIcon} />

### Disabled

Passing te `disabled` prop will apply an opacity to the Button to indicate it is not usable, and disables user interactions with the Button.

<Canvas of={ButtonStories.Disabled} />
