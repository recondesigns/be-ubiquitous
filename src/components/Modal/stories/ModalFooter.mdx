## Basic usage

Import the component, and use it as a child of the `Modal` component. Each button action
requires a value for the `buttonLabel` and `onClick` properties.

```js
import { ModalFooter } from "ubiq-ui"

return (
  <Modal>
    <ModalContent>
      <h3>Modal title</h3>
    </ModalContent>
    <ModalFooter actions={{
      primaryAction: {
        buttonLabel: 'Button',
        onClick: onClickFunc
      },
      secondaryAction: {
        buttonLabel: 'Button',
        onClick: onClickFunc
      }
    }}>
  </ModalContent>
)
```

### Single action button

Providing a single action renders a button.
