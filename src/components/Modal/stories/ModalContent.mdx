## Basic usage

Import the component, and use it as a child of the `Modal` component.

```js
import { ModalContent } from "ubiq-ui"

return (
  <Modal>
    <ModalContent>
      <h3>Modal title</h3>
      <Button>Button</Button>
    </ModalContent
  </ModalContent
)
```
