## Basic usage

Import the component, and use it as a child of the `Modal` component.

```jsx
import { Modal } from "ubiq-ui"

return (
  <Modal>
    <Modal.Content>
      <h3>Modal title</h3>
      <Button>Button</Button>
    </Modal.Content>
  </Modal>
)
```
