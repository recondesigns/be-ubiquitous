## Basic usage

Import the component, and place it in the UI.

```js
import { Progress } from "ubiq-ui"

<Progress percentage={50} completeMessage="Complete!" />
```

### Label

Passing a string to the `label` prop will add a label for the input.
While the Input component does not render the label by default, it is
recommended to include a label with inputs for accessibilty.

<Canvas of={ProgressStories.Default} />

### Required

The optional `required` prop adds the required attribute to the base input element and
and adds an asterisk to the label to indicate the field is required.

<Canvas of={ProgressStories.Default} />
