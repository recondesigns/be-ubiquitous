## Basic usage

Import the component, and place it in the UI.

```js
import { Progress } from "ubiq-ui"

<Progress value={50} completeMessage="Complete!" />
```

### Value

The `value` prop sets the current progress value.

<Canvas of={ProgressStories.Default} />

### Size

Optionally, the component can be give a `size` prop to make it larger, or smaller.

<Canvas of={ProgressStories.Size} />

### Include decimals

To include decimal places in the progress label, pass a number to the `IncludeDecimals` prop.

<Canvas of={ProgressStories.IncludeDecimals} />

Provide more context to users by applying a text string to let users know when the max number has been reached.
