## Basic usage

Import the Dropdown component, provide the options, and place it in the UI.

```jsx
import { Dropdown } from "brutalist-ui"
```

<Canvas of={DropdownStories.Demo} />

### Label

Adds a string as a label for the Dropdown. Can be used without a label, but it is
recommended to use a label.

<Canvas of={DropdownStories.Label} />

### Helper text

Adds a string as a helper text string to the Dropdown for added context for users.

<Canvas of={DropdownStories.HelperText} />

### Has error

Sets error styles on component.

<Canvas of={DropdownStories.HasError} />

### Disabled

Disabled interaction with the component, and adds an opacity to alert the user.

<Canvas of={DropdownStories.Disabled} />

### Max height

Sets the maximum height of the flyout, and allows scrolling.

<Canvas of={DropdownStories.MaxHeight} />

### Close menu when clicking outside component

Closes the menu when the user clicks anywhere outside of the flyout.

<Canvas of={DropdownStories.CloseOnOutsideClick} />

### Close menu using escape

Closes the menu when the user presses the Escape key.

<Canvas of={DropdownStories.CloseOnEsc} />

## Known issues

- When using the `flyoutMaxHeight` prop, keyboard navigation is not scrolling through the options.
