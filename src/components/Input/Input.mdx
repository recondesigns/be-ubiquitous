## Basic usage

Import the component, and place it in the UI.

```js
import { Input } from "ubiq-ui"
```

### Label

Passing a string to the `label` prop will add a label for the input.
While the Input component does not render the label by default, it is
recommended to include a label with inputs for accessibilty.

<Canvas of={InputStories.Label} />

### Helper text

Adding helper text can assist users with the content required for the form.

<Canvas of={InputStories.HelpText} />

### Placeholder

The optional placeholder text defines the required text when the Input has no value.

<Canvas of={InputStories.Placeholder} />

### Disabled

Passing te `disabled` prop will apply an opacity to the Input to indicate it is not usable, and disables user interactions with the component.

<Canvas of={InputStories.Disabled} />

### Errors

Passing te `hasError` prop will apply an error styles to the Input component to indicate there is an error, that or something is not valid.
It is recommended to included helper text describing the error.

<Canvas of={InputStories.Errors} />
