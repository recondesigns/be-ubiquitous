## Getting Started

### Install packages

```sh
npm install be-ubiquitous
```

### Usage

In the consuming app, import the components as needed.

```
import { Button } from 'be-ubiquitous'

export const Form = () => {
  return (
    <form>
      ...otherElements
      <Button label="Submit" />
    </form>
  )
}
```
